Objective : I worked on a Video Game Sales Analysis project where I analyzed historical global sales data to identify trends across platforms, genres, and regions. I used Tableau to build an interactive dashboard showing top-selling games, publisher performance, and regional sales comparisons. The project helped me improve my data cleaning, visualization, and insight-generation skills.

📌 Project Overview

The Video Game Sales Analysis project is a data analytics project where we analyze historical global video game sales data to identify:

Best-selling games
Top platforms
Most successful publishers
Regional sales trends
Genre popularity

📂 Dataset Used

This project used the popular Video Game Sales Dataset from Kaggle.

The dataset usually includes:

Name (Game Title)
Platform (PS4, Xbox, PC, etc.)
Year of Release
Genre (Action, Sports, RPG, etc.)
Publisher
NA_Sales (North America)
EU_Sales (Europe)
JP_Sales (Japan)
Other_Sales
Global_Sales
